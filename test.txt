1. 中缀表达式测试用例 (类型1)：
基本运算：
1 2 + 3 * 4         // 基本四则运算
1 -2 * 3 + 4        // 负数测试
1 2.5 + 3.14 * 4    // 小数测试
复杂运算：
1 2 ^ 3 + 4         // 幂运算测试
1 10 % 3 + 4        // 取模运算
1 5 & 3 | 4         // 位运算测试
函数和常量：
1 s(pi/2)           // 正弦函数和pi常量*
1 c(0) + l(e)       // 余弦函数、对数函数和e常量*
1 t(pi/4) + 1       // 正切函数测试*
括号和优先级：
1 (2 + 3) * 4       // 基本括号
1 ((2 + 3) * 4)     // 嵌套括号
1 [2 + 3] * {4}     // 混合括号类型**
1 8 * {7 + [3 - (2 + 3)] + 2}
1 (1 + 2) * [3 + {4 - 5}]
1 s{pi/2} + c[0]
错误测试：
1 2 + + 3           // 连续运算符错误
1 2 + (3 * 4        // 括号不匹配错误
1 2 + 3 /0          // 除零错误
1 2 3 + 4           // 缺少运算符错误
2. 前缀表达式测试用例 (类型2)：
基本运算：
2 + * 2 3 4         // 基本四则运算
2 + * -2 3 4        // 负数测试
2 + * 2.5 3.14 4    // 小数测试
复杂运算：
2 + ^ 2 3 4         // 幂运算测试
2 + % 10 3 4        // 取模运算
2 | & 5 3 4         // 位运算测试
函数和常量：
2 + s / pi 2 1      // 正弦函数和pi常量*
2 + c 0 l e         // 余弦函数、对数函数和e常量*
2 + t / pi 4 1      // 正切函数测试*
错误测试：
2 + + 2 3           // 运算符参数不足
2 2 + 3             // 格式错误
2 + 3 / 0 4         // 除零错误
3. 后缀表达式测试用例 (类型3)：
基本运算：
3 2 3 4 * +         // 基本四则运算
3 -2 3 * 4 +        // 负数测试
3 2.5 3.14 * 4 +    // 小数测试
复杂运算：
3 2 3 ^ 4 +         // 幂运算测试
3 10 3 % 4 +        // 取模运算
3 5 3 & 4 |         // 位运算测试
函数和常量：
3 pi 2 / s 1 +      // 正弦函数和pi常量
3 0 c e l +         // 余弦函数、对数函数和e常量
3 pi 4 / t 1 +      // 正切函数测试
错误测试：
3 2 3 + + +         // 操作数不足
3 2 3 4 5 +         // 运算符不足
3 3 0 /             // 除零错误
